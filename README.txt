Tech Stack: React.js, Tailwind CSS

Project Overview:
The project includes Login and Signup pages built using React and styled with Tailwind CSS. 
Forms have basic validations (email format, password length, and password confirmation). 
The UI uses a soft pastel theme to keep the interface clean and professional.

How to run locally:
1. unzip the project folder
2. cd aruba-assignment
3. npm install
4. npm start

Notes:
- This project is a frontend demo. There is no backend/auth persistence included.
- For deployment, build with `npm run build` and deploy to Vercel or Netlify.
